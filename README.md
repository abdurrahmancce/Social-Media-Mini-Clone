# 🚀 Pulse — Modern Social Feed UI

<div align="center">

✨ A fully responsive and interactive **Social Media Feed Application** built using **HTML, CSS, and JavaScript** with modern UI/UX design principles.

🌙 Supports Dark Mode • ❤️ Likes & Reactions • 💬 Comments • 🖼 Image Upload • 🔍 Search & Filter

</div>

# ✨ Features

## 🧠 Core Features

- 📝 Create and publish posts
- ❤️ Like & unlike posts
- 😂 Emoji reactions system
- 💬 Comment system
- ✏️ Edit posts
- 🗑 Delete posts with confirmation modal
- 🔍 Live search functionality
- 🎯 Post filtering system
- 🌙 Light/Dark theme toggle
- 🖼 Upload images with preview
- 📱 Fully responsive design
- ⚡ Smooth animations and transitions
- 💾 LocalStorage data persistence

---

# 🎨 UI Highlights

- Modern editorial-style design
- Beautiful typography using **Syne** & **DM Sans**
- Glassmorphism navigation bar
- Animated cards and interactions
- Custom emoji picker
- Trending topics sidebar
- Responsive mobile layout
- Elegant toast notifications

---

# 🚀 Usage

## ✍️ Create Post

- Enter your name
- Write something
- Add emojis or image
- Click **Publish**

---

## ❤️ Interact With Posts

- Like posts
- React using emojis
- Add comments
- Share posts
- Edit/Delete your own posts

---

## 🌙 Theme Switching

Click the ☀ / ☾ button in the navbar to switch themes.
